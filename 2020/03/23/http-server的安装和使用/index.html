解决方案： 解决方案就是在本地自定义服务器，可以通过nodejs、python等任何一个你熟悉的语言来创建本地服务器解决。

http-server介绍

http-server是一个轻量级的基于Nodejs的http服务器，故需要先安装Node.js,http-server可以使项目中的任意一个目录成为服务器的目录,完全抛开后台的沉重工程,直接运行想要的js代码.

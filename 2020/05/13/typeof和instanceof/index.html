typeof和instanceof

概述

​ typeof与instanceof都是在项目中会经常使用到的方法，typeof会返回一个值的类型，而instanceof是用来判断一个对象是否是另一个对象的实例，且只能用来判断对象。

typeof

简介

typeof 操作符用来检测变量的类型，其返回值是一个字符串。

语法

typeof是一个运算符，有2种使用方式：typeof(表达式)和typeof(变量名)，第一种是对表达式做运算，第二种是对变量做运算。

返回值

typeof 的返回值有八个，除了 null 和array都可以显示正确的类型。

  1. undefined ——未定义的变量或值
  2. boolean ——布尔类型的变量或值
  3. string ——字符串类型的变量或值
  4. number ——数字类型的变量或值
  5. object ——对象类型的变量或值，或者null(这个是js历史遗留问题，将null作为object类型处理)
  6. function ——函数类型的变量或值
  7. symbol ——ES6新增的数据格式。 在JavaScript运行时环境中，一个符号类型值可以通过调用函数 Symbol() 创建，这个函数动态地生成了一个匿名，唯一的值
  8. bigInt ——在JavaScript中，BigInt是一种数字类型的数据，它可以表示任意精度格式的整数

注意点

instanceof

简介

typeof 对于对象来说，除了函数都会显示 object，所以说 typeof 并不能准确判断变量到底是什么类型。 如果我们想判断一个对象的正确类型，这时候可以考虑使用 instanceof，因为内部机制是通过原型链来判断的。它是用来判断一个对象是否是另一个对象的实例，注意它只能用来判断对象。

语法

返回值

instanceof的返回值只有两个，分别是true和false。

注意点

对于原始类型来说，你想直接通过 instanceof来判断类型是不行的。
